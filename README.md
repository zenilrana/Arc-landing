# ARC-7 — Cyborg-Themed Landing Page

A responsive landing page concept for a fictional neural-interface implant brand, built for a cyborg-themed UI/UX challenge.

## Overview

ARC-7 imagines a subdermal neural interface product, leaning into the literal "human meets machine" idea rather than generic sci-fi decoration. The centerpiece is a live diagnostic panel where an animated trace morphs between a human heartbeat waveform and a circuit trace, paired with ticking vitals counters (neural sync, pulse, power).

## Features

- Fully responsive layout (desktop, tablet, mobile)
- Animated signature "vitals panel" built with SVG and vanilla JS
- Capability modules grid, spec sheet, and 3-step activation process
- Access request form with client-side validation
- Accessible: visible focus states, respects `prefers-reduced-motion`
- No frameworks or build step — plain HTML, CSS, and JS in a single file

## Tech Stack

- HTML5
- CSS3 (custom properties, CSS Grid, Flexbox)
- Vanilla JavaScript (no dependencies)
- Fonts: Space Grotesk, IBM Plex Sans, IBM Plex Mono (Google Fonts)

## Design Approach

- **Palette:** near-black background with a muted bio-teal accent and amber highlights — clinical and restrained rather than heavy neon
- **Typography:** a technical display face paired with a monospace face for data readouts and specs, reinforcing the "diagnostic system" feel
- **Signature element:** the hero vitals panel, meant to be the one visually memorable moment on the page
